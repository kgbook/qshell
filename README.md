
# qshell
跨平台的终端管理工具，类似 xshell/SecureCRT/mobaxterm 的终端管理工具，目前支持 linux, windows, macos 平台
支持协议：串口，ssh，local-shell

已实现特性：
- 会话管理
- 多标签
- 日记保存
- Button Bar
- Command Window
